## 一、创建菜单页面

我们先分析一下页面布局。

每个页面都会包括一个导航栏区域和内容区域。导航栏（Navbar）是不变的（除了聚焦显示功能），而变化的只有内容（Content）区域。

因此，我们可以为导航栏创建一个组件。

同时，每个页面也需要分别单独写一个组件。需要包括以下页面：
1. PK 页面；
2. Record 页面；
3. Ranklist 页面；
4. User Bots 页面；
5. 404 页面。

bootstrap 地址：[bootstrap 地址](https://v5.bootcss.com/)。

> 注意：bootstrap 的功能可以简单概括为**方便程序员构建美观的页面**。

## 二、创建游戏页面

我们首先实现游戏地图。

游戏地图设计目标如下：地图尺寸为 $13 * 13$；由于两方蛇开始时分别位于地图的左下角和右上角，因此为了公平起见地图需设计为关于左下角到右上角的对角线完全轴对称；地图的最外围一圈为障碍物构成的墙壁，地图内的障碍物随机分布且可手动更改障碍物密度；每次刷新页面后地图中障碍物的布局均会随机刷新；合法的地图被定义为从左下角到右上角需为连通的。
