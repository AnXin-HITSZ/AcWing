## 一、在数据库中创建表 `bot`

表中包含的列：
* `id: int`：非空、自动增加、唯一、主键。
* `user_id: int`：非空。

> 注意：
> 
> 此处，在数据库中的属性名为 `user_id`，在 `pojo` 中需要定义成 `userId`，在 `queryWrapper` 中的名称仍然为 `user_id`。
>
> **总结：** 数据库中的属性若以多单词命名，则建议单词间以下划线分隔，而在 `pojo` 中对应的该属性名应改为驼峰命名法；否则将会报错。

* `title: varchar(100)`。
* `description: varchar(300)`。
* `content: varchar(10000)`。
* `rating: int`：默认值为 `1500`。
* `createtime: datetime`：
  * `pojo` 中定义日期格式的注解：
    * `@JsonFormat(pattern = "yyyy-MM-dd HH:mm:ss")`。
* `modifytime: datetime`：
  * `pojo` 中定义日期格式的注解：
    * `@JsonFormat(pattern = "yyyy-MM-dd HH:mm:ss")`。

## 二、实现后端 API

需要实现的后端 API 如下：
* `/user/bot/add/`：创建一个 Bot。
* `/user/bot/remove/`：删除一个 Bot。
* `/user/bot/update/`：修改一个 Bot。
* `/user/bot/getlist/`：查询 Bot 列表。

## 三、实现前端页面

安装依赖：
* `vue3-ace-editor`；
* `ace-builds`。

示例用法：
```java
import { VAceEditor } from 'vue3-ace-editor';
import ace from 'ace-builds';
ace.config.set(
    "basePath", 
    "https://cdn.jsdelivr.net/npm/ace-builds@" + require('ace-builds').version + "/src-noconflict/")
<VAceEditor
    v-model:value="botadd.content"
    @init="editorInit"
    lang="c_cpp"
    theme="textmate"
    style="height: 300px" />
```
